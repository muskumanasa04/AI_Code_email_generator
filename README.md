AI Email Generator using LangChain
Overview
The AI Email Generator is an intelligent application that automatically generates professional emails based on user input.
It uses Large Language Models (LLMs) integrated through LangChain and Groq to produce structured and context-aware emails.
The system can also process web content and contextual data to generate more relevant responses using vector databases and embeddings.

Features:
Generate professional emails instantly
AI-powered content generation using LLM
Context-based responses using vector database
Web scraping support for additional context
Simple and interactive Streamlit interface
Fast and efficient embedding search with ChromaDB

Technologies Used

Technology	                Purpose
LangChain	        LLM application framework
Groq	            High-speed LLM inference
ChromaDB	        Vector database for embeddings
Sentence          Transformers	Generate text embeddings
Pandas	          Data processing and analysis
BeautifulSoup	    Web scraping
Streamlit        	Web application interface
